# 🔍 Network Port Scanner - Python Cybersecurity Tool

## 📌 Project Overview
A simple  powerful Python-based **port scanner** developed as part of my cybersecurity learning journey. This tool helps identify open ports and common services on target systems, demonstrating practical application of Python in security assessments.

**🎯 Goal**: Apply Python programming skills in real-world cybersecurity scenarios.

## ✨ Features
- ✅ **Common ports scan**: Predefined list of 19 well-known ports (FTP, SSH, HTTP, HTTPS, etc.)
- ✅ **Custom range scan**: Scan any port range you specify
- ✅ **Service detection**: Automatically identifies common services running on open ports
- ✅ **Interactive menu**: User-friendly command-line interface
- ✅ **Cross-platform**: Works on Windows, Linux, and macOS

## 🛠 Technologies Used
- **Python 3.x**
- **Socket Programming** (built-in library)
- **VS Code** - Development environment
  
## 📋 Prerequisites
- Python 3.x installed on your system
- VS Code (optional, any text editor works)
- No external libraries required! (uses only built-in modules)
